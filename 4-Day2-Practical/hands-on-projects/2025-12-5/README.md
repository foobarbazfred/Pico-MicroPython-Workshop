# 二日目演習成果発表  (2025年度版)
２日目は受講者の皆さんに課題を決めていただき、プロトタイプのアプリを開発していただきました。受講者の皆さんの許可を得てこちらにコミットいたします。
- Team01
  - テーマ：[空調コントロールシステム](team01)
  - コンセプト：IoTを活用した環境監視ソリューション、遠隔から空気質を可視化、現場で即時警告
- Team02
  - テーマ：[市田柿メーカ](team02)
  - コンセプト：美味しい干し柿を作る上で温度・湿度の管理が重要、市田柿の干場環境を自動管理し、最適な熟成条件を維持するIoTシステム
- Team03
  - テーマ：[Raspberry Pi Pico を用いた距離・角度可視化システム](team03)
  - コンセプト：ロボットや工作機械が稼働する危険な現場での安全確保が課題、超音波センサとサーボモータを組み合わせた次世代空間認識システム<br>(作業者を中心に360度全方位の安全確保)
